# 🚢 Titanic Survival Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Titanic dataset using Python. The goal is to analyze passenger information, clean the dataset, visualize patterns, and identify the factors that influenced survival.

---

## 🎯 Objective

- Understand the Titanic dataset
- Perform data cleaning and preprocessing
- Handle missing values and outliers
- Explore relationships between features
- Generate meaningful insights using visualizations

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset

The dataset contains passenger details such as:

- Passenger Class
- Age
- Gender
- Fare
- Embarkation Port
- Family Information
- Survival Status

---

## 📊 Exploratory Data Analysis

The following analyses were performed:

- Data Cleaning
- Missing Value Analysis
- Duplicate Check
- Univariate Analysis
- Bivariate Analysis
- Correlation Analysis
- Outlier Detection
- Pair Plot
- Heatmap

---

## 📈 Key Insights

- Most passengers belonged to Third Class.
- Female passengers had a higher survival rate than male passengers.
- First Class passengers survived more frequently.
- Fare contained several high-value outliers.
- Most passengers embarked from Southampton.
- Passenger Class showed a strong relationship with survival.
- Most passengers travelled alone or with small families.
